Release notes - kDrive 3.2.4

New

  • Display of Release Notes in the update popup.
  • The login window is no longer displayed repeatedly if no drive configured.
  • New window for displaying general errors (not linked to a file).
  • Windows: New uninstall option to remove all configuration.


Bug fixes

  • The list of connected drives is not always updated.
  • Block uploading of files with a date too far in the future.
  • db-wal file too big.
  • App crash during synchronization if mouse is in file list area.Â 
  • Windows 11: the color of the app icon now depends on the color of the task bar.
  • Windows 11: the Lite Sync context menu is now directly accessible.
  • macOS: System sound disabled when the app deletes files.
  • macOS: Deletion of placeholder files at the time of desynchronization.
  • macOS Monterey: problem when the app puts files in the recycle bin.
