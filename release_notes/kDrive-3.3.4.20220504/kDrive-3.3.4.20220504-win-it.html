Bug corretti

  • Esclusione dei file non eliminabili dalla sincronizzazione
  • Gestione del log migliorata
  • Crash all’accesso al menu contestuale di file in conflitto
  • Disattivato l’avviso all’aggiunta di un drive già sincronizzato
  • La procedura guidata per l’aggiunta di un drive salta dei passaggi quando si torna alla selezione del drive
  • Crash alla chiusura dell’app durante la sincronizzazione da Finder
  • Problema con l'abilitazione e la disabilitazione di Lite Sync su una cartella con un alias come nome
  • Collegamento sul desktop non eliminato in caso di disinstallazione
  • Blocco dell’aggiornamento de il sistema operativo non è compatibile
  • ARM - estensione non eliminata insieme all’app
