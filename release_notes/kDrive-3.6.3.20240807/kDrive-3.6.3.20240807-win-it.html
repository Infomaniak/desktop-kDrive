  • Corretta la risoluzione dei conflitti di "Move-ParentDelete".
  • Corretto un problema di sincronizzazione con i file i cui nomi sono codificati in NFD.
  • Miglioramento delle prestazioni della gestione dei diritti per i PC su una rete Windows Server
  • Corretto un problema nell'ottenere il percorso della cartella di configurazione se la variabile di configurazione TMP è stata ridefinita
  • Corretto un arresto anomalo durante la rimozione della sincronizzazione
  • Aggiunta protezione per impedire la creazione di sincronizzazioni annidate
  • Archiviazione fissa dei vecchi file di log
  • Correzione dello stile della barra di avanzamento nella procedura guidata di aggiornamento della sincronizzazione
  • Errori corretti" data sempre in formato inglese
