Release notes - kDrive 3.5.8

Bug fixes

  • Fixed a crash in the sync step 3
  • Fixed a crash when aborting a lot of uploads/downloads
  • Fixed the upload session cancellation after an abort
  • Managed connections dropped by kernel under heavy load
  • Fixed an issue when closing an upload session after an error occurred with one chunk
  • Fixed a Move-Delete conflict resolution issue
  • Fixed a DE translation
  • Fixed wrong statuses in Finder when downloading
  • Decreased timeout of long requests (issue with some VPNs)
  • Fixed a crash when a sync is deleted during step 9
