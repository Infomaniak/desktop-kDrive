Release notes - kDrive 3.4.8

Bug fixes

  • Sync failed : move event detected on remote replica but not propagated
  • Filtering of big folders broken, make the app run endlessly
  • Files whose name contains an unsupported character are ignored
  • Moving an item into a blacklisted directory does not trigger new sync
  • Sync started before the end of snapshot generation
  • Safety mechanism to avoid endless loop
  • Folder without write permissions makes the app run endlessly
