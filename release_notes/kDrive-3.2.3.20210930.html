Release note - kDrive 3.2.3

New

  • Possibility to change the language dynamically from the app preferences.
  • Option added in app preferences to put deleted files in recycle bin.
  • Button added to clear the error list.
  • Display of Release Notes at the time of the update.
  • The Preferences window can now be resized.

Bug fixes

  • Login not possible with an alias containing the '+' character.
  • Correction of a graphic issue on the login page.
  • Manage rate limit error (too many requests).
