Release notes - kDrive 3.5.6

Bug fixes

  • Added button to fix all conflicted files at once
  • Fixed login issue while computer is in sleep mode
  • Fixed error message in case of kDrive subscription expiration
  • Fixed crash on step 3
  • Added file size check after download
  • Cleared error list on synchronisation start up
  • Fixed error message in case of LiteSync activation issue
  • Handled error in case of file too big for upload
  • Fixed issue when there were several name clash in the same directory
  • Files containing line return in their name are now ignored
  • Fixed the upload job on file edition
  • Fixed the login window not being displayed if a re-login is requested
  • Disabled the 500Mo folder size limitation by default
