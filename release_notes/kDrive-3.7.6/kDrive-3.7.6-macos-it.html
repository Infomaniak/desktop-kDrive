Note di rilascio - kDrive 3.7.6

Correzioni di bug

  • Corretti i loop di sincronizzazione dovuti alla risoluzione errata dei conflitti EDIT-DELETE.
  • Corretti i loop di sincronizzazione dovuti alle date di creazione nulle delle cartelle.
  • Corretti i cicli di sincronizzazione causati dalla distruzione e dalla ricostruzione di un ramo nella gerarchia delle cartelle di sincronizzazione.
  • Corretto l'aggiornamento dell'elenco delle cartelle escluse dall'utente.
  • Corretti i loop di sincronizzazione causati dalla gestione errata delle date di modifica sui sistemi FAT.
  • Corrette le date di creazione errate impostate dopo la modifica locale.
  • Corrette le azioni sui file (ad esempio, rendere disponibili localmente) i cui nomi contengono caratteri speciali.
  • Corretto il troncamento dei registri.
  • Corretti i loop di sincronizzazione causati da directory create che riutilizzano gli identificatori di elementi eliminati.
  • Corretto un errore per cui i nomi dei file contenenti un carattere di nuova riga non venivano esclusi quando l'utente lo richiedeva.
  • Corretti i loop di sincronizzazione causati da un cambio di directory di sincronizzazione.
  • Visualizzazione di un errore quando il caricamento dei registri non riesce.
  • Corretto il download dei file bundle.
