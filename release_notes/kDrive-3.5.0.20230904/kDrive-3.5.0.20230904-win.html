Release notes - kDrive 3.5.0

Bug fixes

  • Complete overhaul of the synchronization engine:
    • Implementation of an algorithm reflecting the state of science.
    • Communication with the backoffice via the kDrive API instead of Webdav for greater reliability and flexibility in future developments.
    • Massive parallelization to speed up processing and make better use of computer resources.
    • Self-resolution of conflicts and improvement of user information on the choices made.
    • Improved error recovery.
  • Fixed various bugs.
