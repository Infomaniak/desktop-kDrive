Release notes - kDrive 3.2.6

New

  • Synchronization of folders not present when starting the application
  • Management of the creation date of files
  • Uninstalling the application also removes the extension
  • No more double validation needed when selecting large files from the drive tree
  • Possibility to change the login to the drive without resynchronization
  • Synchronization of the drive name if modification from the manager
  • Warning when adding a duplicate drive
  • Files to exclude - no warning by default

Bug fixes

  • Blocking the upload of placeholder files
  • Fix a problem with synchronization with read-only files
  • Optimization of log sending
