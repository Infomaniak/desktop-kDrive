Versionshinweise - kDrive 3.7.7

Fehlerbehebungen

  • Synchronisationsschleifen durch falsche Sortierung von CREATE-Operationen behoben.
  • Synchronisationsfehler behoben, die durch die zunehmende Größe einer hochgeladenen Datei verursacht wurden.
  • Fehler bei der Auswahl und Deselektion von Ordnern im Laufwerksbaum-Widget behoben.
  • Fehlerhafte Erkennung von symbolischen Links behoben.
  • Korrigierte Behandlung von symbolischen Links mit zu vielen Ebenen.
  • Die Behandlung von URLs, die viele Bezeichner von ausgeschlossenen Ordnern enthalten, wurde korrigiert.
  • Das Parsen von komprimierten HTTP-Antworten wurde korrigiert.
