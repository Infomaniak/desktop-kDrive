Release notes - kDrive 3.7.7

Bug fixes

  • Fixed synchronization loops due to incorrect sort of CREATE operations.
  • Fixed synchronization errors caused by the increasing size of an uploaded file.
  • Fixed a bug with the selection and deselection of folders in the drive tree widget.
  • Fixed incorrect identification of symbolic links.
  • Fixed handling of symbolic links with too many levels.
  • Fixed handling of URLs contaning many identifiers of excluded folders.
  • Fixed parsing of compressed HTTP responses.
