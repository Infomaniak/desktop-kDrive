Release notes - kDrive 3.3.6

Bug fixes

  • Renaming a folder for which the user does not have write permissions leads to errors on online files
  • Connexion issue when No Proxy setting is selected
  • Change in trace level not applied on server
  • Advanced synchronization created with LiteSync always deactivated
  • Language in login window does not follow user choice
  • Deactivate high DPI scaling if there is a custom scaling defined in Windows parameters
