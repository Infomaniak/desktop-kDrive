Versionshinweise – kDrive 3.4.4

Fehlerbehebungen

  • Wenn eine neue Version verfügbar ist, bewirkt die Schaltfläche „UPDATE“ nichts
  • Problem beim Kopieren des öffentlichen Links, wenn der Link bereits vorhanden ist
  • Erzeugen Sie eine verständliche Fehlermeldung, wenn der Synchronisierungsordner nicht vorhanden ist
  • Kein Fehler wird in der Benutzeroberfläche angezeigt, wenn der kDrive-Fehler zurückgesetzt wird
  • App zeigt „Synchronisierung auf dem neuesten Stand“, während einige Änderungen nicht übernommen werden
  • Doppelte Löschprüfung: Problem beim Verschieben der Datei
  • Fehler nach dem Verschieben einer großen Anzahl von Dateien
  • In manchen Fällen wird eine MODIFIKATION statt DELETE + CREATE erkannt
  • Der automatische Updater aktualisiert nur kDrive.app, nicht das Deinstallationsprogramm
