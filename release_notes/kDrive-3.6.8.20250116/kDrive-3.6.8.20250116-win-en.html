Release notes - kDrive 3.6.8

New features

  • Allow the user to receive Beta versions of the app.

Optimizations

  • Fine tune some database requests.
  • Keep items path in cache to reduce the number of database requests.

Bug fixes

  • Fixed synchronization issues with items whose name contains `:`.
  • Fixed memory leaks.
  • Enhance handling of file access errors.
  • Fixed a bug where a folder could be synchronized locally but not its content.
  • Fixed updater behavior in case of error.
  • Fixed a crash due to concurrent access.
  • Fixed a bug causing the blacklisting of items.
  • Fixed a migration issue.
  • Fixed a crash when removing a sync before it has started.
  • Handle errors when trashing an item.
