Release notes - kDrive 3.6.3

New features

  • Add the possibility to send log files from the application preferences
  • Activation of automatic testing
  • Improvement of the performance of rights management on Windows Server

Bug fixes

  • Fixed parsing of file name containing special characters such as double quotes, backslashes or line returns
  • Fixed false modification date that causes the application to systematically attempt to re-upload a file
  • Fixed an issue in step 3 that causes the synchronization to restart
  • Fixed issue with ".lnk" files on macOS
  • Fixed issue with upload jobs
  • Fixed the progress bar style while setting up a new synchronisation
  • Fixed frequent abortion of propagation phase
  • Fixed “Move-ParentDelete” conflict resolution
  • Fixed behavior in case of name clash
  • Fixed the way configuration folder path is retrieved on Windows
  • Fixed, on macOS, the access to a link with long path or special character in name
  • Fixed crash on synchronization removal
  • Added protection to prevent the creation of nested synchronizations
  • Fixed issue with files whose names are NFD encoded on Windows
  • Fixed archiving of old log files
  • Update Spanish localization
  • Fixed date format
  • Fixed colors in the conflict resolution window if dark theme activated
